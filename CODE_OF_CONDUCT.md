# Code of Conduct

## Our standard

Contributors are expected to be respectful, technically honest, and careful with safety-sensitive claims. We welcome constructive disagreement, reproducible evidence, and clear communication about uncertainty.

Unacceptable behavior includes harassment, discrimination, personal attacks, deliberate misrepresentation of model results, publishing sensitive data, or proposing unsafe operational claims as established facts.

## Enforcement

Report concerns privately to the repository owner through GitHub. Project maintainers may remove comments, reject contributions, or restrict participation when behavior violates this policy.
