# Dataset manifests

Store one manifest per downloaded dataset here. Do not commit downloaded data. Each manifest should record source name, snapshot/version, acquisition date, checksum, license note, schema version, label definition, time field, entity fields, feature fields, excluded fields, and split strategy.

The planned primary sources are ESA Collision Avoidance Challenge for CDM risk and ESA OPSSAT-AD for telemetry, with NASA SMAP/MSL as the documented fallback.
